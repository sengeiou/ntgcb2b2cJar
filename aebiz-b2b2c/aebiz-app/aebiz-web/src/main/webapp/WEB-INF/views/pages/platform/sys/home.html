1.0.x
版本特性
• 集成Shrio权限框架(支持一级、二级缓存,支持集群部署)
• 集成Redis(SpringMvc缓存及Shiro二级缓存)
• 支持语言国际化(语言包)
• 支持注解式日志(@SLog)
• 支持注解式输出(@SJson)
• 支持自定义路由
• 集成Quartz定时任务(支持集群部署)
• 集成RabbitMq队列(支持集群部署)
• 集成Beetl模板引擎
• 支持Websoket
• 可视化代码生成器
• 采用Bootsrtap+Pjax

使用说明
• 创建空的数据库
• 修改数据库连接 /resources/custom/db.properties
• 项目使用Maven构建，IDEA/Eclipse直接打开，等待包下载完毕即可
• 启动时自动初始化数据表
